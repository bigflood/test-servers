
## install

```
scoop install hadoop-winutils
scoop install spark
pip install pyspark

set PYSPARK_DRIVER_PYTHON=jupyter
set PYSPARK_DRIVER_PYTHON_OPTS='notebook'
```

## run pyspark
```
pyspark
```

## run spark-shell
```
spark-shell
```
